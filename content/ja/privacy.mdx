# プライバシーポリシー

**最終更新日:** 2026年1月8日

Bingify（以下「本サービス」といいます）の運営者（以下「運営者」といいます）は、本サービスを利用するユーザー（以下「ユーザー」といいます）の個人情報の取扱いについて、個人情報の保護に関する法律（以下「個人情報保護法」といいます）その他の関連法令を遵守し、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます）を定めます。

## 1. 定義

本ポリシーにおいて、「個人情報」とは、個人情報保護法第2条第1項に定義される個人情報を指し、生存する個人に関する情報であって、当該情報に含まれる氏名、生年月日その他の記述等により特定の個人を識別することができるもの（他の情報と容易に照合することができ、それにより特定の個人を識別することができることとなるものを含む）、または個人識別符号が含まれるものを指します。

## 2. 収集する情報

### 2.1 ユーザーが提供する情報

- アカウント情報（メールアドレス、ユーザー名等）
- スペース作成データ（スペース名、説明、各種設定情報）
- ビンゴカードの選択および操作履歴
- その他ユーザーが本サービス上で入力、投稿する情報

### 2.2 自動的に収集される情報

- 使用状況データ（アクセス日時、利用機能、操作履歴等）
- デバイス情報（ブラウザの種類、バージョン、OS、IPアドレス等）
- Cookie およびこれに類する技術により収集される情報

### 2.3 外部サービスから取得する情報

本サービスでは、ユーザーの同意に基づき、以下の外部サービスから情報を取得することがあります。

- **YouTube**: チャンネル情報、登録チャンネル情報、メンバーシップ情報等（詳細は後述の「YouTube API Services の利用」を参照）
- **Twitch**: チャンネル情報、フォロー情報、サブスクリプション情報等

## 3. 情報の利用目的

運営者は、収集した情報を以下の目的で利用します。

1. 本サービスの提供、維持、改善および新機能の開発
2. ユーザー認証およびアカウント管理
3. ユーザーサポートおよび問い合わせへの対応
4. 本サービスの利用状況の分析および統計データの作成
5. 不正利用の防止およびセキュリティの確保
6. 利用規約違反行為への対応
7. 本サービスに関する情報の提供
8. その他、上記利用目的に付随する目的

## 4. YouTube API Services の利用

### 4.1 利用の明示

本サービスは、**YouTube API Services** を利用しています。本サービスを利用することにより、ユーザーは以下に同意したものとみなされます。

- [YouTube利用規約](https://www.youtube.com/t/terms)
- [Googleプライバシーポリシー](https://policies.google.com/privacy)

### 4.2 YouTube データの利用

本サービスは、ユーザーの明示的な同意に基づき、YouTube API Services を通じて以下のデータにアクセスします。

- YouTubeチャンネル情報（チャンネルID、チャンネル名等）
- チャンネル登録情報（特定のチャンネルを登録しているかどうか）
- チャンネルメンバーシップ情報（特定のチャンネルのメンバーであるかどうか）

これらの情報は、本サービスの機能提供（ゲートキーパー機能等）のためにのみ使用され、他の目的で使用されることはありません。

### 4.3 アクセス権の取り消し

ユーザーは、[Googleセキュリティ設定ページ](https://myaccount.google.com/permissions)から、いつでも本サービスによるYouTubeデータへのアクセス権を取り消すことができます。

## 5. 外部送信規律（第三者提供先）

本サービスでは、サービスの提供、機能の向上、利用状況の分析等の目的で、以下の外部ツールおよびサービスを利用しており、ユーザーの情報が外部に送信されます。

### 5.1 Google Analytics

- **提供先**: Google LLC
- **送信される情報**: Cookie、アクセスログ、利用状況データ
- **利用目的**: アクセス解析、サービス改善
- **詳細**: [Googleプライバシーポリシー](https://policies.google.com/privacy)

### 5.2 Cloudflare Turnstile

- **提供先**: Cloudflare, Inc.
- **送信される情報**: IPアドレス、ブラウザ情報、操作履歴
- **利用目的**: ボット対策、セキュリティ向上
- **詳細**: [Cloudflareプライバシーポリシー](https://www.cloudflare.com/privacypolicy/)

### 5.3 Supabase

- **提供先**: Supabase, Inc.
- **送信される情報**: ユーザーデータ、コンテンツデータ、認証情報
- **利用目的**: データベース管理、認証基盤の提供
- **詳細**: [Supabaseプライバシーポリシー](https://supabase.com/privacy)

## 6. 情報の共有および第三者提供

運営者は、以下の場合を除き、ユーザーの個人情報を第三者に提供しません。

1. ユーザーの同意がある場合
2. 法令に基づく場合
3. 人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難である場合
4. 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、本人の同意を得ることが困難である場合
5. 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合であって、本人の同意を得ることにより当該事務の遂行に支障を及ぼすおそれがある場合

## 7. データの保存とセキュリティ

1. 運営者は、ユーザーの個人情報を適切に管理し、不正アクセス、紛失、破壊、改ざん、漏洩等の防止のため、必要かつ適切な安全管理措置を講じます。
2. 運営者は、データストレージにSupabaseを使用しており、業界標準のセキュリティ対策を実施しています。
3. ただし、インターネット上での情報送信は100%安全であるとは保証できません。

## 8. リアルタイム通信

本サービスは、接続されているすべてのユーザー間でビンゴカードをリアルタイムで更新するため、WebSocketを使用したリアルタイム同期機能を提供しています。これには、ゲーム状態データの一時的な保存と送受信が含まれます。

## 9. データの保持期間

運営者は、本サービスの提供および本ポリシーに記載された目的を達成するために必要な期間のみ、ユーザーのデータを保持します。ユーザーがアカウントを削除した場合、運営者は合理的な期間内に当該データを削除します。ただし、法令により保存が義務付けられている場合、または紛争解決のために必要な場合は、この限りではありません。

## 10. ユーザーの権利

ユーザーは、個人情報保護法その他の法令に基づき、以下の権利を有します。

1. 個人情報の開示請求
2. 個人情報の訂正、追加または削除の請求
3. 個人情報の利用停止または消去の請求
4. 個人情報の第三者提供の停止の請求

これらの権利を行使される場合は、運営者所定の方法により、運営者までお問い合わせください。

## 11. 子供のプライバシー

本サービスは13歳未満の子供を対象としていません。運営者は、13歳未満の子供から意図的に個人情報を収集することはありません。親権者の方で、お子様が個人情報を提供したと思われる場合は、運営者までご連絡ください。

## 12. 国際的なデータ転送

ユーザーの情報は、ユーザーの居住国以外の国（米国等）に転送され、保存、処理される場合があります。運営者は、適用される法令に従い、適切な保護措置を講じます。

## 13. Cookie およびトラッキング技術

本サービスは、Cookieおよびこれに類する技術を使用して、ユーザーの利便性向上およびサービスの改善を図っています。ユーザーは、ブラウザの設定により、Cookieの受け入れを拒否することができますが、その場合、本サービスの一部の機能が利用できなくなる可能性があります。

## 14. 本ポリシーの変更

1. 運営者は、必要に応じて本ポリシーを変更することができます。
2. 変更後の本ポリシーは、本サービス上に掲載された時点から効力を生じるものとします。
3. 重要な変更を行う場合には、本サービス上での告知その他適切な方法によりユーザーに通知します。

本プライバシーポリシーに関するお問い合わせは、本サービスの[GitHubリポジトリ](https://github.com/ykzts/bingify)を通じてご連絡ください。
